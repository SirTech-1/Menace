# Menace

C++ Chess Engine, UCI compliant


## Board representation
* Bitboards
* Magic bitboards

## Search
* Iterative Deepening
* Transposition Table
* Move ordering
 * MVV/LVA
 * SEE
 * Killer moves
* Selectivity
 * Null Move Pruning
 * Check extensions
 * LMR
 * Quiescence Search

## Evaluation
* Material
* Piece Square table
* Mobility
* Pawn structure
